# react-ts-ocr

A simple app helpful for reading VIN from documents using OCR.

### Example run

```sh
git clone git@github.com:luknp/react-ts-ocr.git
```

```sh
cd react-ts-ocr
```

```sh
npm install
```

```sh
npm start
```
